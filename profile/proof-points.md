# Proof Points - TEMPLATE

Quantified evidence blocks: the specific numbers, benchmarks, and outcomes you can
truthfully cite, plus honest warnings about what is NOT yet confirmed. `cv-rewriter` and
`ats-scanner` pull from here when a CV needs a quantified bullet - they must never invent a
number that isn't recorded here.

Format per item:
```
## <Project / achievement name>

- Claim: TODO - the exact thing you can say
- Evidence: TODO - how you'd back it up if challenged in an interview
- Status: confirmed / not yet quantified - TODO
- Honest framing note: anything a skill should NOT claim yet (e.g. "no speed-up numbers
  until X is measured - do not say 'faster than baseline' until then")
```
